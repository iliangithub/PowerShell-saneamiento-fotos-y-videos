# PowerShell-saneamiento-fotos-y-videos
Básicamente, un programa con 3 funciones, dependiendo de la que elijas hará una cosa u otra.
1) Mostrará todos los archivos de un directorio, que no sean, audio, video o imagen. Para que tu luego manualmente vayas y los borres.
2) Mostrar carpetas vacías.
3) Mostrar archivos y carpetas ocultas.
